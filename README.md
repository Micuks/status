# 📊 服务状态监控

使用 [Upptime](https://upptime.js.org) 监控 micuks.click 的各项服务。

## 监控的服务

- 🎨 配色推荐器 (palette.micuks.click)
- 🖼️ 看看腿 (kankan.micuks.click)
- 📈 AI 投资组合 (investment.micuks.click)
- ♟️ 象棋服务
- 🔌 学术配色后端 API
- 🔌 AI 投资后端 API
- ⚫ 五子棋 (gomoku.micuks.click)
- 🧱 俄罗斯方块 (tetris.micuks.click)

## 工作原理

Upptime 使用 GitHub Actions 每 5 分钟检查一次服务状态：

- ✅ 服务正常：记录响应时间
- ❌ 服务异常：自动创建 Issue 告警
- 📊 生成历史数据和图表

## 状态页面

访问 [status.micuks.click](https://status.micuks.click) 查看实时状态。
